View all admins:  
- Shows a list of all admins registered.  
- Columns: name, email.  

Manage medicine:  
- Shows a list of all medicines.  
- Columns: ID, name, shape, colour, dosage, quantity, updated.  
- Update medicine stock by changing quantity, then save.  

Advanced >> create new admin:  
- Enter first name, last name, username, password, email, then sign up.  

Advanced >> view all patients:  
- Shows a list of all patients registered.  
- Columns: name, ..., assigned admin.  
  - Assigned admin: admin name / null.  
  - Change assigned admin: select from all admins.  
- Toggle to present alternative view: view unassigned patients.
